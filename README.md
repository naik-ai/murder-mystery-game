# Murder Mystery Game

A physical murder mystery party game designed for immersive social gatherings.

## Overview

This project contains all the materials needed to host an engaging murder mystery party where players take on character roles, gather clues, and work together to solve a fictional crime.

## Features

- **Character Profiles**: Detailed backstories and secret information for each player
- **Clue Cards**: Physical evidence pieces to discover throughout the game
- **Timeline Events**: Structured game flow with reveals at key moments
- **Host Guide**: Complete instructions for running the game smoothly
- **Printable Materials**: Ready-to-print character sheets, clue cards, and props

## Getting Started

### What You'll Need

- 6-12 players (varies by scenario)
- A suitable venue (home, event space, etc.)
- Printed game materials
- Optional: costumes, decorations, themed refreshments

### Setup

1. Choose a scenario
2. Print all required materials
3. Assign characters to players
4. Set up the play area with clue locations
5. Brief players on the rules

## Project Structure

